Hello! Welcome to Handibotting ...

First, to help you finish unpacking we'll move the Z-axis — it's the up-down, vertical axis that moves the router — to a higher position so you can remove the protective foam cylinder.

The tool will bang into its upper stop to make sure it is all the way up (this doesn't do any damage). So, close the lid, then click the green bar just below…

Click to Move the Z-axis up a little bit for clearance*

*Note that you’ll always be asked for permission whenever the Handibot is about to move. To OK motion, push and hold (1 sec) the plastic Green Button on the top of the tool.

After the router has moved up, remove that protective foam tube.

Moving Your Handibot Around…

You use the KEYPAD to move the head of your Handibot around to get it in position for cutting, changing bits, or out-of-the-way.

Click on "Target" in upper right corner to toggle the KEYPAD in and out. Use the arrow buttons to move the Handibot around. If you move the Handibot too far in one direction of another, it will bang into its stops. Despite the noise, it is safe to move the Handibot to all extreme directions. Just beware of moving the router all the way down, the installed bit can crash into your material, damaging the bit. Also note that if you run into the end-of-travel in any direction, the location of the tool with respect to your "Home" point will be lost.

Each time you use your Handibot, set its Home Location.

Set Your XYZ HOME LOCATION
Clear area around the tool first!

Use this Green Button to get setting the Home Location started. Homing will be done for all 3 axes, so make sure the area is clear for your Handibot to move around.

Note: if you manually (ie. using the KEYPAD) move the Handibot position and run into an end-stop after homing XYZ, you will have to set the XYZ home location again. The function to Set the Home Location is also included in the FireUp! App. The FireUp! will be useful for daily set-up tasks. It also contains a more detailed explanation of Setting Home and the special case of setting the Z-axis after a bit change.

NOW, let's try running a job…

Your Handibot should be all set to run a test cut file on the board that came attached to the bottom of your tool. We'll run the same job on the lower half of the board that was run in the top of the work area before it left Handibot HQ.

To run this project, we will be making use the Job Manager system-app which you will normally access from the left-side panel when you are ready to run a job.

Job Preparation:

  • Switch the router ON, and ensure the speed is set to 3 on the router dial.
  • If you have a vacuum handy you can hook it up to the outlet on the Handibot, but we won't be making a lot of dust.
  • Close the hood.

Run the "Test-Carve" Job
to bring up the Job Manager

Press the green arrow in the Job Manager.

Press the Green Button on the top of the tool.

As the file runs, note that there is a STOP indicator at the bottom of the window. Hit this Stop, or the flashing red square if you need to Stop the file!

If you haven't already done it, please take a look at the Handibot Safety Section from the Handibot Manual. [You should have received a hard copy with your tool. If not, there is a PDF version on the web at docs.handibot.com; but since you are probably connected to your tool in AP Mode at the moment, you'll need to get back on the Internet in order to view a copy.]

Upon completion of the cut, hit target button and use keypad to move the Handibot to the rear.

After completing the test cut, relaunch the Quick Launch App and click on the Next Steps tab.

 

 

Next Steps in Working with Handibot

The Dashboard provides access for all the work you will be doing with your Handibot. Note its System Apps in the left-side panel, the App Area in the center, and the right-side, DRO panel with Keypad for manually moving the tool-head.

Handibot Utility Apps

Most Apps will provide a direct way to carry out specific jobs or tasks. Hole Cutter is an example of a basic utility App.

More Advanced Apps

On-board Apps can also carry out some simple parametric design tasks. Check out Beamer and TerrainMan. Basically set the parameters and then click "Done" or "Cut" and your project will be sent to the Job Manager to run.

Specialty Apps

  • Touch-and-Go will move your tool-head to a designated location.
  • Sb4 is a traditional CNC interface, similar to ShopBot's standard interface.
  • Fire-Up! will help you with daily Handibot set-up activities.
  • theNews will keep you posted on the latest Handibot News and Bulletins.
  • ExampleApp is the definitive example of how to create an App.

Putting Your Handibot on a Wi-Fi Network

So far, you've been exploring Handibotting using the direct Access Point (AP) Mode, wireless link to the tool. You can also put Handibot/FabMo on your Wi-Fi network in order to give it access to jobs and resources in the cloud or over your network. For instructions, put your device back on the internet and go to: docs.handibot.com in order to access the: "Joining a Network" document.
