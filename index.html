Hello! Welcome to Handibotting ...

First, to help you finish unpacking we'll move the Z-axis — it's the up-down or vertical axis that moves the router — to a higher position so you can remove the protective foam cylinder.

The tool will bang into its upper stop for a bit to make sure it is all the way up (this doesn't do any damage). So, close the lid, then click the green bar just below…

Click to move the
Z-axis up for clearance*

*Note that you’ll always be asked for permission whenever the Handibot is about to move. To OK motion, push and hold (1 sec) the plastic Green Button on the top of the tool.

After the router has moved up, remove that protective foam tube.

Moving Your Handibot Around…

You use the KEYPAD to move the head of your Handibot around to get it in position for cutting, changing bits, or just to get it out-of-the-way.

Click on "Target" in upper right corner of the screen to toggle the KEYPAD in and out. Use the arrow buttons to move the Handibot around. If you move the Handibot too far in one direction or another, it will bang into its stops. Despite the noise, it is safe to move the Handibot into these extremes. You won't hurt anything. Just beware of moving the router all the way down as the installed bit can crash into your material, damaging the bit.

Each time you use your Handibot, be sure to set its Home Location.

Set Your XYZ HOME LOCATION
Clear area around the tool first!

Use this Green Button set your Home Location now. Setting the home locations will be done for all 3 axes, so make sure the area is clear for your Handibot to move around.

Note: if you manually move the Handibot using the KEYPAD and run into an end-stop after setting home for an axis, you will have to set the XYZ home locations again. The function to Set the Home Location is included in the Fire-Up! App. The Fire-Up! App will be useful for daily set-up tasks.

NOW that you've got it all figured out, let's try running a job…

Your Handibot should be all set to run a test cut file on the board that came attached to the bottom of your tool (hopefully it is still bolted on). To test everything out we'll run the same job on the lower half of the board that was run in the top part of the work area before it left Handibot HQ.

To run this project, we will be making use the Job Manager system-app which you will normally access from the left-side panel when you are ready to run a job.

Job Preparation:

  • Switch the router to ON, and ensure the speed is set to 3 on the router dial.
  • We'll be making some sawdust, so if you have a shop-vac or vacuum cleaner handy you should hook it up to the outlet on the Handibot.
  • Close the hood.

 

Just a couple last things before running the test cut:

.
  • After you start by clicking the Green bar below, a green arrow will appear in the Job Manager which you will need to click to start the "Job".
  • And, as before, you'll also need to press the Green Button on the top of the tool to authorize the action.
  • As the file runs, note that there is a STOP indicator at the bottom of the window. Hit this Stop, or the plastic Red Button on the Handibot if you need to Stop the file!
  • After completing the test cut, relaunch this Quick Launch App and click on the Next Steps tab.

Next Steps in Working with Handibot

If you haven't already done it, please take a look at the Handibot Safety Section from the Handibot Manual. [You should have received a hard copy of the Safety Section with your tool. If not, there is a PDF version on the web at docs.handibot.com; but since you are probably connected to your tool in AP Mode at the moment, you'll need to get back on the Internet in order to view a copy.]

Also, as you have now finished running the test file, it would be a good time to remove the test panel from the bottom of your Handibot -- if you have not already done so. There are two bolts holding the panel on. Removing the panel will give your Handibot full access for machining surfaces and materials below it. You can always make convenient jigs and fixtures using a variation of this method for attaching your tool to a board. And, if you have not already seen it, check out the "Accessory Base" in the Handibot Store. It provides an enclosed work area for small projects.

The Dashboard

The Dashboard provides access for all the work you will be doing with your Handibot. Note its System Apps in the left-side panel, the general App Area in the center, and the right-side, DRO panel with Keypad for manually moving the tool-head.

 

Handibot Utility Apps

Most Apps will provide a direct way to carry out specific jobs or tasks. Hole Cutter is an example of a basic utility App.

More Advanced Apps

On-board Apps can also carry out some simple parametric design tasks. Check out Beamer and TerrainMan. Basically set the parameters and then click "Done" or "Cut" and your project will be sent to the Job Manager to run.

Specialty Apps

  • Smooth Sketch do quick and fun drawings and carve them into a surface.
  • Touch-and-Go will move your tool-head to a designated location.
  • Sb4 is a traditional CNC interface, similar to ShopBot's standard interface.
  • Fire-Up! will help you with daily Handibot set-up activities.
  • theNews will keep you posted on the latest Handibot News and Bulletins.
  • ExampleApp is the definitive example of how to create an App.

Putting Your Handibot on a Wi-Fi Network

So far, you've been exploring Handibotting using the direct Anchor Point (AP) Mode, a direct wireless link to your tool. You can also put Handibot/FabMo on your Wi-Fi network to give it access to jobs and resources on your network or in the internet cloud. If you don't have instructions handy, you can get them by putting your device back on the internet and going to: docs.handibot.com in order to access the: "Hooking-Up: Network mode" document.
