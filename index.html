Hello! Welcome to Handibotting ...

We’re going to move the tool and carve a test file.

We'll move the Z-axis &emdash; it's the up-down, vertical axis that moves the router &emdash; to a higher position. The tool will bang into the upper limit (but it won't do any damage.) So, close the lid, then click the next green label…

Move the Z-axis up a little bit for clearance

You’ll always be asked for permission whenever the Handibot is about to move, to concur, push and hold (1 sec) the plastic Green Button on the top of the tool.

After the router has moved up, remove that protective foam tube.

Moving Your Handibot Around…

Click on target in upper right corner and use the arrow buttons to move the Handibot. Despite the noise, it is safe to move the Handibot to all extreme directions. Just beware of moving the router all the way down, the installed bit can crash into your material, damaging the bit.


Each time you use your Handibot, start by setting the Home location for each axis.

CLICK HERE > to Set Your XYZ HOME LOCATION << Clear area before clicking !

And yes, you'll need to use the Green Button again to get things started. Note that you can always use this "Set XYZ Home" function when needed to make sure the coordinates are all calibrated as they should be. This same function is included in the FireUp! App that will be useful to you for daily set-up tasks. We're making sure we are exactly "homed" here in all axes before running the test job.

NOW, let's try running a job ...

Your Handibot should be all set to run a test file on the board that came attached to the bottom of your tool. We'll run the same job on the lower half of the board that was run in the top of the work area before it left Handibot HQ. It was our check to make sure your tool was working well.

Job Preparation:

  • Make sure the switch on the router is set to ON, and that the speed is set to about 3 on the dial on the router.
  • If you have a vacuum handy you can hook it up to the outlet on the Handibot, but we won't be making a lot of dust.
  • To run this project, we will be making use the Job Manager system-app which you will normally access from the left-side panel when you are ready to run a job.

After you've run this first job ... you will have hopefully gotten the basic idea of how a job is done with a Handibot. After it's done, if you come back to this App and move on to the Next Steps tab above, we'll introduce you to where you will want to go from here.

You'll first go to the "Start" panel for the Job Manager. Just click the green arrow to get the job started when you are ready. You'll need to provide authorization with that Green Button. As the file runs, note that there is a STOP indicator at the bottom of the window. Hit this Stop, or the flashing red square if you need to Stop the file!

Next Steps in Working with Handibot

The Dashboard provides access for all the work you will be doing with your Handibot. Note its System Apps in the left-side panel, the App Area in the center, and the right-side, DRO panel with Keypad for manually moving the tool-head.

Handibot Utility Apps

Most Apps will provide a direct way to carry out specific jobs or tasks. Hole Cutter is an example of a basic utility App.

More Advanced Apps

On-board Apps can also carry out some simple parametric design tasks. Check out Beamer and TerrainMan. Basically set the parameters and then click "Done" or "Cut" and your project will be sent to the Job Manager to run.

Specialty Apps

  • Touch-and-Go will move your tool-head to a designated location.
  • Sb4 is a traditional CNC interface, similar to ShopBot's standard interface.
  • Fire-Up! will help you with daily Handibot set-up activities.
  • theNews will keep you posted on the latest Handibot News and Bulletins.
  • ExampleApp is the definitive example of how to create an App.

Putting Your Handibot on a Wi-Fi Network

So far, you've been exploring Handibotting using the direct Access Point (AP) Mode, wireless link to the tool. You can also put Handibot/FabMo on your Wi-Fi network in order to give it access to jobs and resources in the cloud or over your network. For instructions, put your device back on the internet and go to: docs.handibot.com in order to access the: "Joining a Network" document.
